PQL Searches

You can use PQL (Phoenix Query Language) to construct simple or complex searches against metadata fields.

Examples

The following query search for all media files where the genre contains horror or thriller
Genre contains 'Horror' or Genre contains 'Thriller'
The following query search for all media files where the MediaType has not been set
MediaType is null
You can also group conditions and use and or or to create a more complex query. The following query is basically what BMT uses to find missing metadata
MediaType is null or MediaTitle is null or (MediaType = 'TV' and EpisodeNumber = 0)
If you enter a search with only 1 or 2 words then that search will be treated a Title = 'Search Criteria'. i.e., the follow query will search for all shows with a Title of House
House

Notes

  • FieldName are case sensitive
  • and and or is case sensitive
  • The following operands are supported: =, >, <, <>, is null, is not null, contains
  • The complete list of Metadata is not nicely documented but you can decipher the Field Names from this file.
